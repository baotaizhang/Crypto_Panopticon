Bitcoin/NEM Node Checker

You can request like following.By clicking node you can see the recipients.

下記のようにurlに記載することで、BTC/NEMの送金履歴を可視化します。nodeをクリックすることで、アドレスを確認可能です。

http://160.16.197.85:3000/? + address = [your address] + & call = [number of recursive calling]

Cf BTC. http://160.16.197.85:3000/?address=14WSZksP2RKkAPZbedgqc3YVa5Lp82weLn&call=4

Cf NEM. http://160.16.197.85:3000/?address=NC4C6PSUW5CLTDT5SXAGJDQJGZNESKFK5MCN77OG&call=10&nemtrace=true
