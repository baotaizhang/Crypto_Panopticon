# Crypto_Panopticon

---

**Crypto_Panopticon** is an implementation of the platform to detect thieves.
Still in a beta state.

## Install

```
$ git clone https://github.com/YusukeShimizu/Crypto_Panopticon.git
$ cd Crypto_Panopticon
$ npm install
$ node candy.js
```
